1. Shareable Campaign URL
When the cart has at least 1 item, show a small "Share campaign" button next to the floating cart button, or inside the cart modal header area.
On click, encode the full cart state into the URL as a query string parameter and copy it to clipboard. Format:
?cart=ChannelName:qty:pin,ChannelName:qty
Example: ?cart=CryptoVault:2:pin,BlockTalk:1,LennyCrypto:1
:pin suffix means Homepage Pin is active for that channel.
On page load, check for a ?cart= parameter. If present, parse it and hydrate the cart state automatically — restore quantities, Homepage Pin toggles, card selected states, sticky banner, and cart badge count. All silently, no modal or announcement.
After copying, show a small inline toast notification near the button: "Campaign link copied!" in green, fades out after 2.5 seconds. No alert dialogs.
